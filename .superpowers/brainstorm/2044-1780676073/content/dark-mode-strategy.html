Dark Mode Strategy

Your design is dark-first (dark backgrounds, gold accent). How should light mode work?

A

Dark Only

No light mode. The dark palette IS the brand. SSENSE, many luxury brands do this — dark is inseparable from the identity. Simpler to maintain, stronger brand consistency.

B

Dark Default, Light Available

Dark is primary, but offer a light mode toggle for readability preference. Light mode uses warm cream/ivory backgrounds (#faf8f5) with the same brass accent. More inclusive, more work to maintain.

C

Light Default, Dark for Key Sections

Main browsing is light/cream for readability. Hero, featured sections, and checkout go dark for drama. Best of both — readable product pages but editorial impact where it counts. Like Mr Porter's approach.
